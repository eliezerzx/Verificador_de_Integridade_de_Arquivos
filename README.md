# Verificador_de_Integridade_de_Arquivos
O código tem como objetivo verificar a integridade de arquivos por meio do cálculo do hash SHA-256 de cada um. Esse tipo de verificação é usado para garantir que os arquivos não foram alterados ou corrompidos desde sua última verificação.

# Resumo:
O script serve para auditar e garantir a autenticidade de arquivos, sendo útil em contextos de segurança da informação, controle de versões e verificação de integridade de dados.
